### 1. Commit Message Structure

> 타입(스코프) : 주제(제목)

 (줄바꿈 필수)

> 본문

### 2. Comit Type
|type|Context|
|----|-------|
|feat|새로운 기능|
|fix|버그 수정|
|build|빌드 관련 파일 수정|
|doc|문서 수정|
|style|코드 스타일 혹은 포맷 변경|
|refactor|코드 리펙토링|
|test|테스트 코드 수정|
|perf|성능 개선|
|etc|그외 수정사항|
